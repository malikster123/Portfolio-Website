AWS To the Power Of

The game "To the Power Of" allows users to find any number's power. The following technology stack was used in the development of this web application:
AWS Amplify: To develop and host the webpage.
AWS Lambda: To write the games function using python.
AWS Gateway: To invoke the Lambda function.
DynamoDB: to store the application's results
AWS IAM: Robust permission management for Lambda execution roles..
